Instructions

This tool helps you identify unused or missing images from the images folder that is associated with a LiveLabs Markdown file that you specify. The tool:

  1. Lists all extra (unused) images in the images folder that is associated with the your lab's .MD file that you select. The unused images are displayed on the right-hand side of the page.
  2. Lists any images that are referenced in the lab (.MD) file that you select which are missing from the images folder. The missing images are displayed on the left-hand side of the page.

To use this tool select the:

  • Markdown (MD) file for a LiveLabs lab that you want to check (on the left-hand side).
  • Images folder that is associated with the MD file that you selected (on the right-hand side).

To check the images for a new lab, simply select the new MD file and the associated images folder.
